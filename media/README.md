\# Media and Demonstration Materials



This folder contains lightweight visual material associated with the augmented reality teleoperation architecture for Xiaomi CyberDog.



The media resources provide complementary evidence of the system operation, including the AR teleoperation interface, RGB/depth visual feedback, and the experimental setup used during implementation and validation.



\## Purpose



This folder supports the practical understanding of the AR teleoperation workflow through visual material related to the implemented system.



The content is focused on demonstration and methodological clarity rather than manuscript formatting or publication layout.



\## Folder structure



```text

media/

├── screenshots/

│   ├── ar\_hud\_rgb\_view.png

│   ├── ar\_hud\_depth\_view.png

│   └── experimental\_setup.png

│

└── demo\_links/

&#x20;   └── demo\_video\_links.md

```



\## Media content



The folder may contain visual material associated with:



```text

AR HUD running on Meta Quest 3

Unity teleoperation interface

CyberDog RGB camera feedback

CyberDog depth-enhanced feedback

Operator interaction with the AR interface

Experimental setup with Meta Quest 3, Ubuntu Bridge PC, and Xiaomi CyberDog

```



\## Demonstration videos



Demonstration videos are referenced through external links to keep the repository lightweight.



External video links are organized in:



```text

media/demo\_links/demo\_video\_links.md

```



\## File formats



The repository uses lightweight and accessible media formats:



```text

PNG  screenshots and interface captures

JPG  compressed visual material

GIF  short lightweight demonstrations

```



Large video demonstrations are stored externally and referenced through links.



\## Naming convention



Media files use descriptive names that identify the content and context of each resource.



Examples:



```text

ar\_hud\_rgb\_view.png

ar\_hud\_depth\_view.png

metaquest\_ar\_interface.png

cyberdog\_experimental\_setup.png

control\_channel\_test.png

visual\_feedback\_test.png

```



\## Privacy scope



The media material included in this folder does not contain participant-identifiable information.



Images and videos associated with user evaluations are limited to non-sensitive technical or demonstrative content related to the teleoperation system.
