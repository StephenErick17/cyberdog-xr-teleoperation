# Media

This folder contains a selected collection of photos, screenshots, and videos obtained during the development, testing, and validation stages of the CyberDog XR teleoperation project.

The files included here are provided as complementary visual material. Their purpose is to document part of the experimental process, the system setup, the Unity interface, the robot operation, and several tests carried out throughout the project.

This folder does not contain the complete set of images or videos recorded during the research. Instead, it includes representative excerpts selected to help illustrate how the system was developed, tested, and validated.

## Important note

Downloading this folder is not required to use the repository.

The media files are not necessary to run the source code, review the implementation, or use the processed experimental data. They are included only as optional visual support for users who want to better understand the experimental setup, the interface behavior, and the operation of the teleoperation system.

Users who only need the code, datasets, or documentation can skip downloading this folder.
